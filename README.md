# py-quick-share-server

Wi-Fi経由でデバイス間のファイルを即座にやり取りするための、超軽量ファイル転送サーバーです。

## 概要

「PCにあるファイルをスマホに送りたい」「スマホの写真をPCに保存したい」という作業を、ケーブルやクラウドサービスを使わず、ブラウザだけで完結させます。Pythonがインストールされている環境であれば、どこでも一瞬でファイル交換拠点になります。

## 主な機能

- **双方向転送**: PCからのダウンロードだけでなく、ブラウザからのアップロードに対応。
- **QRコード生成**: 起動時に接続用URLをQRコードでターミナルに表示。スマホでのURL入力が不要です。
- **OS不問**: ブラウザさえあれば、iPhone, Android, Mac, Windows間でやり取り可能。
- **セキュリティ**: ローカルネットワーク内でのみ完結するため、外部サーバーにデータが流出しません。

## 使い方

1. サーバを起動します。

```bash
python main.py
```

2. ターミナルに表示されたQRコードをスマホでスキャンします。
3. ブラウザが開いたら、ファイルをアップロードまたはダウンロードしてください。

## 注意事項

- 本ツールは信頼できるローカルネットワーク内（家庭内Wi-Fiなど）での使用を想定しています。
- 公衆Wi-Fiなどの環境では、同じネットワーク内の他人にファイルが見える可能性があるため、使用を控えてください。
